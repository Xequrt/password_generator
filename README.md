# Генератор паролей

Генератор паролей с проверкой надёжности. Реализованы две основные функции: генерация по заданным параметрам и оценка сложности пароля.

## Возможности

- Генерация пароля заданной длины из настраиваемого набора символов:
  - строчные буквы (всегда включены);
  - прописные буквы (по флагу);
  - цифры (по флагу);
  - спецсимволы (по флагу).
- Детерминированная генерация: один и тот же `seed` всегда даёт одинаковый пароль.
- Проверка надёжности пароля по 5 критериям с вердиктом и оценкой от 1 до 5.

## Как работает

`generate_password(length, seed, use_uppercase=True, use_digits=True, use_special=False)`

- Генерирует пароль длины `length`.
- Набор символов формируется по флагам.

Примеры:

```python
generate_password(8, seed=1)
# => "fzxc0WG0"

generate_password(12, seed=123, use_special=True)
# => "vfJ0p2rH&j9#"

generate_password(5, seed=42, use_uppercase=False, use_digits=False)
# => пароль только из строчных букв
```

## Функция `check_password(password)`

Оценивает пароль по 5 признакам:
- Длина ≥ 8 символов.
- Наличие строчной буквы.
- Наличие прописной буквы.
- Наличие цифры.
- Наличие спецсимвола.

Возвращает строку вида:
- "Слабый пароль (оценка 2 из 5)",
- "Средний пароль (оценка 3 из 5)" и т. д.

Пример:

```python
check_password("Abcdef123!")
# => "Очень надёжный пароль (оценка 5 из 5)"

check_password("abc123")
# => "Слабый пароль (оценка 2 из 5)"
```

## Запуск

Создайте файл `main.py` рядом с `password_generator.py`.

Добавьте туда тестовые вызовы функций (как в примерах выше).

Запустите в терминале:

```bash
python main.py
```
